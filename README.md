## :arrow_forward: How to Run App

1. cd to the repo
2. Run Build for either OS
  * for iOS
    * run `react-native run-ios`
  * for Android
    * Run Genymotion
    * run `react-native run-android`
  * for Web
    * Next.js : run `npm run next:dev`
    * Webpack : run `npm run webpack:dev`
